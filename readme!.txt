Сначала нужно проинсталлировать nodejs на ваш linux, в данной работе была установалена Nodejs v10.16.0 на Debian GNU/Linux 9 (stretch).
Затем файлы index.js, server.js, router.js, requestHandlers.js, requestHandlers2.js, requestHandlers3.js, requestHandlers4.js, requestHandlers5.js расположить в удобном для вас каталоге.
После чего для удобства перейти в этот каталог и запустить систему веб-сервисов командой: nodejs index.js
Далее в вашей локальной сети будет запущена система веб-сервисов, проверить работу можно на любом устройстве с браузером внутри этой сети, зайдя в браузер и прописав: https://localhost:8888/
Приятной работы!
